# SHOGIMAN (Web Prototype)

ファミコン風UI + 爆風エフェクトで可動範囲を表示する将棋プロトタイプです。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## 操作例（爆風表示）

1. 先手の飛車（初期位置: 8筋2段目相当）をクリック
2. 上下左右に十字状の爆風タイルが連なって表示
3. 赤系タイル（敵駒取得可能）と通常移動タイルを見分ける
4. 任意の可動マスをクリックして移動

## 実装済み内容

- 9x9 将棋盤
- 先手/後手の初期配置
- 先手駒をクリックして合法手を表示
- 駒ごとの爆風風レンジ演出
- 移動と捕獲
- RESET で初期化
- `CPU STATUS: STANDBY (DUMMY)` 表記
